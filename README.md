# Getting started

- install ruby 2 rubyinstaller-devkit-2.7.5-1-x64.exe from https://rubyinstaller.org/downloads/
````
gem install jekyll bundler
bundle install
````


# Use

Build and hot reload.
````
bundle exec jekyll serve --watch
````
